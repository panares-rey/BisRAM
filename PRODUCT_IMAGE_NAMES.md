## Notes
- Recommended image size: **600×600px** or larger, square ratio
- If an image file is missing, the site will automatically show a default microchip icon placeholder
- Supported format: `.jpg` (JPEG). Keep file sizes under 300KB for fast loading.
- These images are seeded only on **first run** (when the database is empty). If the DB already exists, update product image URLs via the Admin panel.
